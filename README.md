# MUSE - SEO & Content SLM
Le pilier "Accessible". Responsable de la clarté et du contenu SEO.

## Rôle
- Rédaction de fiches produits SEO.
- Synthèse des informations trouvées sur le web par ATLAS.
- Mise en forme des citations.

## Installation & Usage
```bash
uv run main.py
```

## Dataset (dans /data)
Contient des exemples de styles de rédaction et de formats de citations.
